# Overview
Multi-threaded element queue project (C++/Cmake)

Author: Manuel Ferreira

# 1. Files

* Queue Project
    * queue_project.tpp
    * queue_project.hpp
    * CMakeLists

* GTest
    * queue_project_gtest.cpp

* Example implementation
    * main.cpp

# 2. Build
```bash
mkdir build
cd build/
cmake ..
make
```

# 3. Run Example
```bash
./QueueExample
```

# 4. Run GTests
```bash
./QueueTests
```
